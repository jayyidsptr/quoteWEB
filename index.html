Dokumentasi API Kutipan

Gunakan API ini untuk mendapatkan kutipan berdasarkan kategori yang berbeda. Berikut adalah cara menggunakan API:

  • GET /api/quotes/:category - Mendapatkan kutipan berdasarkan kategori. Kategori yang tersedia adalah:
    • life - Kutipan tentang kehidupan
    • love - Kutipan tentang cinta
    • motivation - Kutipan motivasional

Contoh: Mengakses /api/quotes/life akan mengembalikan kutipan tentang kehidupan.

Pilih kategori dan tekan tombol untuk melihat kutipan.
